# Репозиторий для общей документации по REST API

Перед началом работы установите утилиту tsp:

```bash
$ npm install -g @typespec/compiler
```

Чтобы скомпилировать файл схемы, можно использовать:

```bash
$ npm run build
```

Для запуска и просмотра получившейся документации:

```bash
$ npm start
```

Также оно всё уложено в докер, но это скорее для более удобного развёртывания, для разработки не слишком удобно.
